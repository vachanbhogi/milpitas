# Phonics & Speech Impediment Helper (Whisper WASM)

This is a Vite + React + TypeScript project created for a hackathon. The goal of the project is to use **Whisper (via WASM / WebGPU)** to detect a child's voice, teach them phonics, and assist them against speech impediments entirely in the browser.

## Tech Stack
- **Framework:** React 19 (TypeScript)
- **Build Tool:** Vite 8
- **Package Manager:** Bun
- **Speech Recognition:** Whisper WASM (Transformers.js / ONNX Runtime Web)

## Getting Started

To install the dependencies:
```bash
bun install
```

To start the development server:
```bash
bun run dev
```
